The background of this web application

It all started one summer when a friend of mine introduced me to a radio website (web application, to be specific) called LISTEN.moe

I thought to myself, "how hard is it really to create a website like this?", so I tried to replicate it by myself and this is the result. To everyone looking through my code, when I started working on this I had next to no knowledge about HTML, CSS or JavaScript and as a result that's why the code design is a bit messy. I'll probably have to look through everything and clean + refactor if I want to fix that, haha (I finally know why programmers hate messy code :sob:)

About this web application

This is a web application that functions as a radio website. It plays a raw audio stream provided by LISTEN.moe and gets JSON data about the playing song's name, artist, duration etc. from their WebSocket. For more details about the inner workings, you can visit their documentation page.

I don't claim any copyright and neither is it in my intention to profit from this work. I made this for educational purposes and private use, if there are any problems please contact me so we can work it out.
